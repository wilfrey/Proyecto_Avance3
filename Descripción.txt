La pagina web esta creada con LocalStorage para las reservaciones y con sessionStorage los correos.Debido a que si creaba 2 localstorage la informacion de cada
uno se iva a añadir a las tabla. Por lo tanto, para que no chocaran entre si utilice localStoage & sesssionStorage.

El JSON fue utilizado para mostrar los datos de los creadores de la pagina web.
